## Context
This is the **Reconciliation details** modal in Synder — where a user configures a Transaction Reconciliation run. They've already selected:
- **Accounting platform:** QuickBooks Online ("My lovely money")
- **Integration:** Shopify ("My ceramic plates")
- **Automation mode:** Automated (Synder retrieves Shopify data for them)
- **Date range:** 02/22/2024 – 03/22/2024

The user is now stuck on the **Clearing account** dropdown (required field), and the dropdown is open showing three sections.

## The Dropdown (exact content)

**Section 1: "ACCOUNTS AVAILABLE FOR AUTOMATED DATA RETRIEVAL"**
- Shopify (required for Synder)

**Section 2: "OTHER ACCOUNTS CREATED BY SYNDER"**
- Shopify (required for Synder): Shopify Manual Order (required for Synder)
- Shopify Other Order (required for Synder)

**Section 3: "OTHER QBO ACCOUNTS"**
