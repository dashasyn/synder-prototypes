Top Values

KEY INSIGHT

"Winning one accounting firm = winning all their clients."

Justin Quintero sets up Synder for every new Shopify client the firm takes on. TJ Cariaga recommends to every client with 1,000+ transactions/month.

Section 5 — Hesitations

Objections & Hesitations

💰 PRICE
"The price just doubled on us... I'm trying to ride it out."— Tony Jamal (but stayed)
"I cannot justify the cost for the second business."— Soroush Hatami
Price objection = expansion, not renewal. Primary account stickiness is strong.
🔧 COMPLEXITY
"I thought I could do it myself... it just kind of blew me away."— Vic Henderson
"Your interface — I think it's complicated, it's not intuitive."— David Nozick
Many users are using Synder below its capability — they don't know features exist.
🔒 TRUST
"I have almost zero confidence right now that our books are correct."— Kyle Gibson
"My reports don't mean anything if I can't reconcile them."— David Nozick
🎯 Reconciliation = the trust-builder. Lead with it in every demo.
What almost made users leave: Slow RevRec processing at month-end · Persistent unresolved bugs · Duplicate/double data · Being told "your setup is wrong" by support
Feature Language

Smart Rules & RevRec — Value Framing

⚡ Smart Rules

Users call them: "automations," "rules," "customizations"

"Being able to set my own automation rules — that's one of the best features."— Clifton McMorris
⚠️ Many users aren't using all their rules — Tony Jamal: "I haven't leveraged them yet. Maybe I need to educate myself."
✅ Frame as: "automation you control"

📊 RevRec

For SaaS users: not a feature — it's THE reason they signed up.

"Ever since we started with RevRec, I'm glad Synder handles that, not me. I was doing it on spreadsheets."— Brenton Fowler
🎯 Many Stripe users don't know RevRec exists. Demo it explicitly — show upgrades, downgrades, cancellations handled automatically.
