## Minsk Bus Stop Poster Redesign Project (ongoing)

### Context
- Ignat is working on redesigning Minsk public transit bus stop information posters
- This is a personal/civic design project, NOT Synder-related
- Previous session (2026-03-16): built a transit scheme builder in `/bus-schemes/` — generates SVG route trees from CSV data
- Current version: v10 (generate-preview-v10.js) — added small white dots at stops instead of tick marks

### Key Design Decisions (discussed today)
1. **Two-color system:** Yellow (#F5A623) = bus, Teal (#4CAF50) = trolleybus — replaces the current 8-10 color chaos
2. **Poster layout:** Schedule cards on LEFT, route scheme on RIGHT (London-inspired)
3. **Scheme style:** Moscow-inspired vertical column layout, NOT the comb/tree from v9
   - Shared trunk runs vertically, branches split into parallel columns
   - Stop names horizontal, readable — not tiny circles like current Minsk
   - Small white dots on route lines at each stop
   - Route numbers: small at branch points, big badges at terminus
   - Metro stops: bold + Ⓜ icon
   - Routes can split and rejoin (merge logic needed)
4. **Physical size:** A1 poster
5. **Max routes per stop:** 14

### Problems to Solve
- Split-and-rejoin topology (routes diverge then share stops again)
- Column duplication — same stop names repeating across branches
- Space management with 14 routes branching
- Route tracking without individual colors (use frequent route number labels every 4-5 stops)

### Reference Screenshots Analyzed
- Current Minsk posters: 4 screenshots showing inconsistent layouts, too many colors, tiny unreadable stop names
- London TfL spider maps: gold standard — consistent template, colored routes, readable text
- Moscow: single-color vertical column with route badges, street map panel
- Berlin BVG: single route per poster, connecting services shown
