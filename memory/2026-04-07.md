## Lessons from this session
- **Playwright + MUI custom grids**: `table`/`th`/`tr` selectors don't work — must use `data-testid`, role attrs, or walk up from an interactive element
- **Playwright forced click on elements behind tooltips**: use `{ force: true }` to bypass the tooltip overlay intercept
- **Dynamic hover tooltips (MUI)**: must dismiss with `mouse.move(1,1)` between each capture or tooltips stack
- **InfoOutlinedIcon `data-testid` trick**: gives you a stable selector for MUI info icons
- **Storage state rotation**: when `grails_remember_me` expires but CF_Authorization still valid, a fresh Playwright login works by using the existing state as baseline then filling the Synder form (selector: `input[placeholder="email@mail.com"]` — NOT `type=email`)
