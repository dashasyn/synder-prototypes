  - **Accounting company** = the software (QBO, Xero) — NOT "accounting platform"
  - **Books** = the data/ledger — NOT "your company", "your accounting"
  - **Enable/Disable** — NOT "turn on/off"
  - **Click** — desktop only, NOT "press" or "tap"
  - **Add** — standard verb for creating items (not "Create")
  - **Higher plan** — for upsells (not specific plan names)
  - **Numbers** — always digits (3, not "three")

## Toast Pattern Defined
- Success/enable/change: `{Thing} {past participle}!` (exclamation)
- Disable: `{Thing} disabled.` (period, neutral)
- Failure: `Couldn't {verb} {noun}.` (period, calm)
- NO "was successfully" — ever
