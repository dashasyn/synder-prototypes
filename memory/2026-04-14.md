## Morning Idea — UX Pattern from Another Product
- **Topic:** Triage "one at a time" mode for Transaction Reconciliation Not Matched tab
- **Angle:** UX pattern (Linear's triage inbox + Superhuman inbox-zero framing)
- **Idea:** Add a "Review one at a time" mode alongside the existing table. Shows one unmatched item with full context + three actions: Match / Ignore / Skip. Keyboard navigation. "All reviewed" when done. Power users keep the table; new users get the guided path. No backend changes.
- **Built on:** April 10 prototype — 44 unmatched items as a flat list felt overwhelming; inbox-zero framing converts a pile into a completable task
